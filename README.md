# Site Certeiro

A página que faz o cliente escolher você. Landing page do serviço de páginas profissionais de [Alessandro Ribeiro](https://alessandroribeiro.com.br).

Site estático em um único arquivo (`index.html`) — sem build, sem dependências. É só servir.

Acessível por padrão (leitor de tela e teclado): antes de mexer no HTML/CSS/JS, leia o [ACESSIBILIDADE.md](ACESSIBILIDADE.md) — tem o que já foi feito e o checklist pra não regredir.

## Rodar localmente

Abra o `index.html` no navegador, ou:

```sh
npx serve .
```
